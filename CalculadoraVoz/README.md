# CalculadoraVoz

* Dentro del analisis de señales se puede realizar una comparativa en funcion de su FFT. Esto nos permite comprobar que palabra, letra se pronuncio. En base a eso, se presenta este pequeño software que funciona como una calculadora donde el ingreso de la informacion es utilizando la voz.
* Su funcionamiento es basico, nada avanzado. Se debe tener una base de datos de la voz par eso se creo una funcion que se encarga de automatizar ese proceso.
* Actualmente se elimino esa base de datos por lo que si se desea usar o comprobar se debe comenzar por crearla.
